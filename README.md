## –ö–æ–Ω—Ç—Ä–æ–ª—å–Ω—ã–µ –≤–æ–ø—Ä–æ—Å—ã
### 1. –ß—Ç–æ —Ç–∞–∫–æ–µ Clang, –∏ –∫–∞–∫–æ–≤–∞ –µ–≥–æ —Ä–æ–ª—å –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏ –ø—Ä–æ–≥—Ä–∞–º–º?
Clang ‚Äî —ç—Ç–æ —Ñ—Ä–æ–Ω—Ç–µ–Ω–¥-–∫–æ–º–ø–∏–ª—è—Ç–æ—Ä –¥–ª—è —è–∑—ã–∫–æ–≤ C, C++, Objective-C –∏ –¥—Ä., –≤—Ö–æ–¥—è—â–∏–π –≤ –ø—Ä–æ–µ–∫—Ç LLVM. –ï–≥–æ –∑–∞–¥–∞—á–∞ ‚Äî –∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å –∏—Å—Ö–æ–¥–Ω—ã–π –∫–æ–¥, —Å—Ç—Ä–æ–∏—Ç—å –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω–æ–µ —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–æ–µ –¥–µ—Ä–µ–≤–æ (AST), –ø—Ä–æ–≤–æ–¥–∏—Ç—å —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–∏–π –∏ —Å–µ–º–∞–Ω—Ç–∏—á–µ—Å–∫–∏–π –∞–Ω–∞–ª–∏–∑, –∞ –∑–∞—Ç–µ–º –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤—ã–≤–∞—Ç—å –∫–æ–¥ –≤ –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ LLVM IR, –∫–æ—Ç–æ—Ä–æ–µ –∑–∞—Ç–µ–º –ø–µ—Ä–µ–¥–∞—ë—Ç—Å—è –≤ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä LLVM –¥–ª—è –¥–∞–ª—å–Ω–µ–π—à–µ–π –æ–±—Ä–∞–±–æ—Ç–∫–∏ (–æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∏ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –º–∞—à–∏–Ω–Ω–æ–≥–æ –∫–æ–¥–∞).

### 2. –ß—Ç–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è–µ—Ç —Å–æ–±–æ–π LLVM –∏ –∫–∞–∫ –æ–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã—Ö –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞—Ö?
LLVM (Low Level Virtual Machine) ‚Äî —ç—Ç–æ –∏–Ω—Ñ—Ä–∞—Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞, –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—é—â–∞—è –∏–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç—ã –¥–ª—è –∞–Ω–∞–ª–∏–∑–∞, –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∏ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –º–∞—à–∏–Ω–Ω–æ–≥–æ –∫–æ–¥–∞. –û–Ω–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ –∫–∞—á–µ—Å—Ç–≤–µ –±—ç–∫–µ–Ω–¥–∞ –≤–æ –º–Ω–æ–≥–∏—Ö –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞—Ö (–≤–∫–ª—é—á–∞—è Clang) –∏ —Å–æ—Å—Ç–æ–∏—Ç –∏–∑:LLVM IR ‚Äî –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–æ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è, –æ–ø—Ç–∏–º–∏–∑–∏—Ä—É—é—â–µ–≥–æ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–∞, –≥–µ–Ω–µ—Ä–∞—Ç–æ—Ä–∞ –º–∞—à–∏–Ω–Ω–æ–≥–æ –∫–æ–¥–∞.
–°–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–µ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä—ã –∏—Å–ø–æ–ª—å–∑—É—é—Ç LLVM –¥–ª—è –ø–µ—Ä–µ–Ω–æ—Å–∏–º–æ—Å—Ç–∏, –≥–∏–±–∫–æ—Å—Ç–∏ –∏ –º–æ—â–Ω–æ–π –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∫–æ–¥–∞.

### 3. –ß–µ–º –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –∞–±—Å—Ç—Ä–∞–∫—Ç–Ω–æ–µ —Å–∏–Ω—Ç–∞–∫—Å–∏—á–µ—Å–∫–æ–µ –¥–µ—Ä–µ–≤–æ (AST) –æ—Ç –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–æ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è LLVM IR?
AST (Abstract Syntax Tree) ‚Äî —ç—Ç–æ –≤—ã—Å–æ–∫–æ—É—Ä–æ–≤–Ω–µ–≤–∞—è —Å—Ç—Ä—É–∫—Ç—É—Ä–∞, –æ—Ç—Ä–∞–∂–∞—é—â–∞—è –≥—Ä–∞–º–º–∞—Ç–∏—á–µ—Å–∫—É—é —Å—Ç—Ä—É–∫—Ç—É—Ä—É –∏—Å—Ö–æ–¥–Ω–æ–≥–æ –∫–æ–¥–∞. –û–Ω —Å–æ–¥–µ—Ä–∂–∏—Ç –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ —Ç–∏–ø–∞—Ö, —Å—Ç—Ä—É–∫—Ç—É—Ä–∞—Ö, –æ–ø–µ—Ä–∞—Ç–æ—Ä–∞—Ö –Ω–∞ —É—Ä–æ–≤–Ω–µ —è–∑—ã–∫–∞.
LLVM IR (Intermediate Representation) ‚Äî —ç—Ç–æ –Ω–∏–∑–∫–æ—É—Ä–æ–≤–Ω–µ–≤–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ, –±–ª–∏–∑–∫–æ–µ –∫ –º–∞—à–∏–Ω–Ω–æ–º—É –∫–æ–¥—É, –Ω–æ –≤—Å—ë –µ—â—ë –∞–±—Å—Ç—Ä–∞–≥–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ –æ—Ç –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–π –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä—ã. –û–Ω–æ –ø—Ä–µ–¥–Ω–∞–∑–Ω–∞—á–µ–Ω–æ –¥–ª—è –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∏ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∫–æ–¥–∞.
–ü—Ä–∏–º–µ—Ä: a + b –≤ AST ‚Äî —ç—Ç–æ –±–∏–Ω–∞—Ä–Ω–∞—è –æ–ø–µ—Ä–∞—Ü–∏—è –Ω–∞ —É—Ä–æ–≤–Ω–µ C++, –∞ –≤ LLVM IR —ç—Ç–æ –º–æ–∂–µ—Ç –±—ã—Ç—å add i32 %a, %b.

### 4. –î–ª—è —á–µ–≥–æ –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –ø—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ (IR) –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ –∫–æ–º–ø–∏–ª—è—Ü–∏–∏?
–ü—Ä–æ–º–µ–∂—É—Ç–æ—á–Ω–æ–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ (IR) –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –¥–ª—è: —É–Ω–∏—Ñ–∏–∫–∞—Ü–∏–∏ –æ–±—Ä–∞–±–æ—Ç–∫–∏ –∫–æ–¥–∞, –Ω–µ–∑–∞–≤–∏—Å–∏–º–æ –æ—Ç –∏—Å—Ö–æ–¥–Ω–æ–≥–æ —è–∑—ã–∫–∞,–æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –∫–æ–¥–∞ (–Ω–∞ —É—Ä–æ–≤–Ω–µ IR –ø—Ä–æ—â–µ –ø—Ä–∏–º–µ–Ω—è—Ç—å –ø—Ä–µ–æ–±—Ä–∞–∑–æ–≤–∞–Ω–∏—è),–≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∫–æ–¥–∞ –¥–ª—è —Ä–∞–∑–Ω—ã—Ö –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä –±–µ–∑ –∏–∑–º–µ–Ω–µ–Ω–∏—è —Ñ—Ä–æ–Ω—Ç–µ–Ω–¥–∞,–æ–±–ª–µ–≥—á–µ–Ω–∏—è –∞–Ω–∞–ª–∏–∑–∞ –ø—Ä–æ–≥—Ä–∞–º–º—ã.
### 5. –ß—Ç–æ –¥–µ–ª–∞–µ—Ç –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏—è alloca –≤ LLVM IR, –∏ –∑–∞—á–µ–º –æ–Ω–∞ –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –≤ —Ñ—É–Ω–∫—Ü–∏—è—Ö?
alloca ‚Äî —ç—Ç–æ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏—è LLVM IR, –∫–æ—Ç–æ—Ä–∞—è –≤—ã–¥–µ–ª—è–µ—Ç –ø–∞–º—è—Ç—å –Ω–∞ —Å—Ç–µ–∫–µ –≤–Ω—É—Ç—Ä–∏ —Ñ—É–Ω–∫—Ü–∏–∏. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –ª–æ–∫–∞–ª—å–Ω—ã—Ö –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö. –ü—Ä–∏–º–µ—Ä:

–æ–∑–Ω–∞—á–∞–µ—Ç –≤—ã–¥–µ–ª–µ–Ω–∏–µ 4 –±–∞–π—Ç –ø–æ–¥ int –Ω–∞ —Å—Ç–µ–∫–µ.

### 6. –ó–∞—á–µ–º –Ω—É–∂–Ω–∞ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏—è –∫–æ–¥–∞ –≤ –∫–æ–º–ø–∏–ª—è—Ç–æ—Ä–µ, –∏ –∫–∞–∫–∏–µ –æ—Å–Ω–æ–≤–Ω—ã–µ —Ü–µ–ª–∏ –æ–Ω–∞ –ø—Ä–µ—Å–ª–µ–¥—É–µ—Ç?
–û–ø—Ç–∏–º–∏–∑–∞—Ü–∏—è –Ω—É–∂–Ω–∞ –¥–ª—è:

*—É–º–µ–Ω—å—à–µ–Ω–∏—è —Ä–∞–∑–º–µ—Ä–∞ –∫–æ–¥–∞,
*—Å–Ω–∏–∂–µ–Ω–∏—è —ç–Ω–µ—Ä–≥–æ–ø–æ—Ç—Ä–µ–±–ª–µ–Ω–∏—è (–∞–∫—Ç—É–∞–ª—å–Ω–æ –¥–ª—è –≤—Å—Ç—Ä–∞–∏–≤–∞–µ–º—ã—Ö —Å–∏—Å—Ç–µ–º),
*—É–¥–∞–ª–µ–Ω–∏—è –Ω–µ–∏—Å–ø–æ–ª—å–∑—É–µ–º–æ–≥–æ –∫–æ–¥–∞ –∏ —É–ø—Ä–æ—â–µ–Ω–∏—è –ª–æ–≥–∏–∫–∏.

### 7. –ß—Ç–æ —Ç–∞–∫–æ–µ SSA-—Ñ–æ—Ä–º–∞ –∏ –ø–æ—á–µ–º—É –æ–Ω–∞ –≤–∞–∂–Ω–∞ –ø—Ä–∏ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏ –ø—Ä–æ–≥—Ä–∞–º–º?
SSA (Static Single Assignment) ‚Äî —ç—Ç–æ —Ñ–æ—Ä–º–∞ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è –∫–æ–¥–∞, –≤ –∫–æ—Ç–æ—Ä–æ–π –∫–∞–∂–¥–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è –ø—Ä–∏—Å–≤–∞–∏–≤–∞–µ—Ç—Å—è —Ç–æ–ª—å–∫–æ –æ–¥–∏–Ω —Ä–∞–∑, –∏ –Ω–æ–≤—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –ø–æ–ª—É—á–∞—é—Ç –Ω–æ–≤—ã–µ –∏–º–µ–Ω–∞. –≠—Ç–æ:
—É–ø—Ä–æ—â–∞–µ—Ç –∞–Ω–∞–ª–∏–∑ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–µ–π,–ø–æ–∑–≤–æ–ª—è–µ—Ç –±–µ–∑–æ–ø–∞—Å–Ω–æ –∏ —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ –ø—Ä–æ–≤–æ–¥–∏—Ç—å –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏,–æ–±–ª–µ–≥—á–∞–µ—Ç –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏–µ –≥—Ä–∞—Ñ–æ–≤ –¥–∞–Ω–Ω—ã—Ö.

### 8. –ß—Ç–æ —Ç–∞–∫–æ–µ –≥—Ä–∞—Ñ –ø–æ—Ç–æ–∫–∞ —É–ø—Ä–∞–≤–ª–µ–Ω–∏—è (CFG) –∏ –∫–∞–∫ –æ–Ω –ø–æ–º–æ–≥–∞–µ—Ç –∞–Ω–∞–ª–∏–∑–∏—Ä–æ–≤–∞—Ç—å –ø–æ–≤–µ–¥–µ–Ω–∏–µ –ø—Ä–æ–≥—Ä–∞–º–º—ã?

–û–Ω –ø–æ–º–æ–≥–∞–µ—Ç:
–æ–±–Ω–∞—Ä—É–∂–∏—Ç—å –º–µ—Ä—Ç–≤—ã–π –∫–æ–¥,–ø—Ä–æ–≤–æ–¥–∏—Ç—å –∞–Ω–∞–ª–∏–∑ –¥–æ—Å—Ç–∏–∂–µ–Ω–∏—è –ø–µ—Ä–µ–º–µ–Ω–Ω—ã—Ö,—Ä–µ–∞–ª–∏–∑–æ–≤—ã–≤–∞—Ç—å —Ü–∏–∫–ª—ã, –≤–µ—Ç–≤–ª–µ–Ω–∏—è –∏ –∏—Å–∫–ª—é—á–µ–Ω–∏—è,–æ—Å–Ω–æ–≤–∞ –¥–ª—è –º–Ω–æ–≥–∏—Ö –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–π (–Ω–∞–ø—Ä–∏–º–µ—Ä, loop unrolling, inlining).

### 9. –ö–∞–∫ —É—Å—Ç—Ä–æ–µ–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏–µ –∞—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏—Ö –æ–ø–µ—Ä–∞—Ü–∏–π –≤ LLVM IR (–Ω–∞–ø—Ä–∏–º–µ—Ä, —É–º–Ω–æ–∂–µ–Ω–∏–µ, —Å–ª–æ–∂–µ–Ω–∏–µ)?
–ê—Ä–∏—Ñ–º–µ—Ç–∏—á–µ—Å–∫–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω—ã –∫–∞–∫ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏. –ü—Ä–∏–º–µ—Äs:

### 10. –ü–æ—á–µ–º—É —Ñ—É–Ω–∫—Ü–∏–∏ –≤ LLVM IR –æ–±—ã—á–Ω–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å–æ–±–æ–π –æ—Ç–¥–µ–ª—å–Ω—ã–µ –µ–¥–∏–Ω–∏—Ü—ã –∞–Ω–∞–ª–∏–∑–∞ –∏ –æ–ø—Ç–∏–º–∏–∑–∞—Ü–∏–∏?

### 11. –ß—Ç–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å —Ñ—É–Ω–∫—Ü–∏–µ–π –≤ LLVM IR, –µ—Å–ª–∏ –æ–Ω–∞ –≤—ã–∑—ã–≤–∞–µ—Ç—Å—è –æ–¥–∏–Ω —Ä–∞–∑ –∏ –æ—á–µ–Ω—å –∫–æ—Ä–æ—Ç–∫–∞—è?

## 8. –î–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω–æ–µ –∑–∞–¥–∞–Ω–∏–µ: –∏—Å—Å–ª–µ–¥–æ–≤–∞–Ω–∏–µ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è —Å—Ç—Ä—É–∫—Ç—É—Ä—ã (–≤–∞—Ä–∏–∞–Ω—Ç¬†2)
